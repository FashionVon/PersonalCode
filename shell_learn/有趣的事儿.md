**10月11号，在郑州办了人才引进，让我3个月去高人才查询准迁证有没有出来,顺手写了一个脚本每天检测。**

```shell
#! /bin/bash

url="http://www.zzrc.cn/publicSearch/zqzResultSearch.do?name='姓名'&idCardNo='身份证号'&resultType='准迁证'";
html=`curl -v -s $url 2>&1 | grep -c 对不起`

start_stamp=`date -d 20171011 +%s`
now_stamp=`date +%s`

daydiff=`expr \( ${now_stamp} - ${start_stamp} \) / 86400`

if [[ $html -ge 1 ]]
then
    echo '继续等，准迁证一定快出来了!' | mail -s "准迁证的最新消息,已经过了${daydiff}天" liuzhifeng01@58ganji.com
else
    echo '恭喜你，准迁证出来了!' | mail -s "准迁证的最新信息,已经过了${daydiff}天" liuzhifeng01@58ganji.com
fi
```
